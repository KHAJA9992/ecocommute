Daily Commute Tracking App
Overview
A web application that helps users track their daily commute patterns, understand their environmental impact, and receive suggestions for more sustainable transportation choices.

Core Features
Transport Mode Selection
Users can select from predefined transport modes: bus, car, auto, bicycle, walking, and other common options
Allow users to enter distance traveled for each transport mode daily
Support multiple transport modes for a single day's commute
Tracking and Analytics
Track daily distance traveled by each transport mode
Calculate and display cumulative distance traveled over time
Store historical commute data for trend analysis
Environmental Impact Calculation
Calculate estimated carbon emissions for each transport choice based on distance and mode
Display positive environmental impact when users choose eco-friendly options (public transport, walking, cycling)
Show greenhouse gases avoided compared to car usage
Provide daily and cumulative CO₂ savings metrics
Sustainability Suggestions
Suggest walking or cycling for shorter commute distances (configurable threshold)
Show nearby walkable routes for short-distance commutes
Provide personalized recommendations to improve sustainability based on user's travel patterns
Dashboard Interface
Clean, dashboard-style interface displaying:
Daily travel summary by transport mode
CO₂ emissions and savings
Environmental impact metrics
Sustainability suggestions
Visual charts or graphs showing travel patterns over time
Progress indicators for environmental goals
Data Storage
The backend must store:

User commute entries (date, transport mode, distance)
Historical travel data for analytics
User preferences and settings
Carbon emission factors for different transport modes
Backend Operations
Save daily commute entries
Retrieve user's travel history and analytics
Calculate carbon emissions and environmental impact
Generate sustainability suggestions based on travel patterns
Provide route suggestions for walkable distances
